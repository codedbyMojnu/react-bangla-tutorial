# Web Development শেখার জন্য কম্পিউটারে পরিবেশ তৈরি করুন

Web Development শিখতে গেলে আগে কিছু জিনিস আমাদের কম্পিউটারে সেটআপ করে নিতে হবে। আমি এখানে একদম সহজভাবে বলছি, যেন আপনি খুব সহজেই আপনার কম্পিউটারে ওয়েব ডেভেলপমেন্ট শেখার জন্য পরিবেশ তৈরি করে নিতে পারেন। 

---

## ১. Node.js ইনস্টল করুন

JavaScript কোড লোকাল মেশিনে রান করানোর জন্য Node.js ইনস্টল করা প্রয়োজন। এটি এমন একটি সফটওয়্যার যা আপনার কম্পিউটারকে একটি JavaScript রানটাইম এনভায়রনমেন্টে পরিণত করে। অর্থাৎ, Node.js ইনস্টল করলে আপনি চাইলে নিজের কম্পিউটারেই একটি সার্ভার চালাতে পারবেন অথবা ক্লায়েন্ট-সাইড কোডও লোকালভাবে টেস্ট করতে পারবেন। 

এটা ছাড়া React প্রজেক্ট রান হবে না।

- [Node.js ওয়েবসাইটে](https://nodejs.org/) যান।
- LTS ভার্সন ডাউনলোড করে ইনস্টল করুন।

ইনস্টল হয়েছে কিনা দেখতে টার্মিনালে লিখুন:

```bash
node -v
npm -v
```

দুইটা ভার্সন নাম্বার দেখলে বুঝবেন, আপনার কম্পিউটারে Node.js ঠিকভাবে ইনস্টল হয়েছে।

---

## ২. একটি কোড ইডিটর ইনস্টল করুন (VS Code ইনস্টল করুন)

কোড লেখার জন্য **VS Code Editor** বর্তমান সময়ে খুবই জনপ্রিয়। এটা ফ্রি, আর অনেক ফিচার আছে। **Extensions** অপশন থেকে আপনি প্রয়োজনীয় এক্সটেনশনগুলো ইনস্টল করে নিতে পারবেন। যা আপনার **ডেভেলপমেন্ট এক্সপেরিয়েন্স** অনেকগুণে বাড়াবে। 

- [VS Code ওয়েবসাইটে](https://code.visualstudio.com/) যান।
- ডাউনলোড করে ইনস্টল করুন। 

দেখুন, আপনার কম্পিউটারের টার্মিনাল থেকে `code .` কমান্ড কাজ করে কি'না? কমান্ডলাইন সম্পর্কে ধারণা রাখুন, এটি আপনার _ডেভেলপমেন্ট স্পিড_ বাড়ায়। `code .` কমান্ড কাজ না করলে, `ctrl + shift + p` চাপ দিয়ে path দিয়ে সার্চ করুন এবং `Shell command: install 'code' command from PATH এ ক্লিক করুন।

## ৩. পছন্দের থিম ব্যবহার করুন

VS Code-এ সুন্দর থিম থাকলে চোখের আরাম হয়। এক্সটেনশন থেকে পছন্দের থিম ইনস্টল করে নিন। 

যেমন: LWS VS Code Theme।

- এক্সটেনশন ট্যাবে যান (Ctrl+Shift+X)।
- "LWS VS Code Theme" লিখে সার্চ দিন, ইনস্টল করুন। 

> এই থিমে অনেক কালার `variant` আছে। আপনি Learn with Sumit- Peace of the eye কালার ভেরিয়েন্টটি ব্যবহার করতে পারেন।

## ৪. React অ্যাপ তৈরি করুন (Vite দিয়ে)

আগে create-react-app ব্যবহার হতো,  
এখন Vite দিয়ে খুব দ্রুত React প্রজেক্ট বানানো যায়।

টার্মিনাল খুলে লিখুন:

```bash
npm create vite@latest my-react-app -- --template react
cd my-react-app
npm install
npm run dev
```

এখন ব্রাউজারে `http://localhost:5173` খুললে আপনার অ্যাপ দেখবেন।

## ৫. `.js` আর `.jsx` ফাইল

React কম্পোনেন্ট বানাতে `.jsx` এক্সটেনশন ব্যবহার করলে ভালো হয়।  
এতে কোড পরিষ্কার বোঝা যায়।

## ৬. কোড ফরম্যাট আর linting

- Prettier দিয়ে কোড অটো ফরম্যাট হবে।
- ESLint দিয়ে ভুল ধরতে পারবেন।

VS Code-এ এই এক্সটেনশন দুইটা ইনস্টল করুন।

---

## ৭. VS Code সেটিংস

VS Code-এর সেটিংসে নিচের কনফিগারেশন দিন:

```json
{
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.formatOnSave": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  }
}
```

---

## ৮. দরকারি এক্সটেনশন

| এক্সটেনশন         | কাজ                 |
| ----------------- | ------------------- |
| ESLint            | ভুল ধরবে            |
| Prettier          | কোড ফরম্যাট করবে    |
| Path Autocomplete | import সাজেস্ট করবে |
| Auto Rename Tag   | ট্যাগ rename সহজ    |

## ১০. Git আর GitHub

কোডের পরিবর্তন ট্র্যাক করতে Git ব্যবহার করুন।  
GitHub-এ কোড রাখলে হারিয়ে যাবে না।

- [Git ডাউনলোড করুন](https://git-scm.com/downloads)।
- টার্মিনালে গিয়ে লিখুন:

```bash
git init
git add .
git commit -m "Initial commit"
```
