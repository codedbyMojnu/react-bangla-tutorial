# Synthetic Events Reference / সিনথেটিক ইভেন্ট রেফারেন্স

React এর SyntheticEvent সিস্টেম native DOM events এর একটি wrapper যা সব browsers এ consistent behavior প্রদান করে।

## Quick Reference / দ্রুত রেফারেন্স

### Mouse Events / মাউস ইভেন্ট

| Event | Purpose / উদ্দেশ্য | When Triggered / কখন ট্রিগার হয় |
|-------|---------------------|-----------------------------------|
| `onClick` | Click event | Mouse click or Enter key |
| `onDoubleClick` | Double click | Two rapid clicks |
| `onMouseDown` | Mouse button press | Mouse button pressed |
| `onMouseUp` | Mouse button release | Mouse button released |
| `onMouseMove` | Mouse movement | Mouse moves over element |
| `onMouseEnter` | Mouse enter | Mouse enters element |
| `onMouseLeave` | Mouse leave | Mouse leaves element |

### Keyboard Events / কীবোর্ড ইভেন্ট

| Event | Purpose / উদ্দেশ্য | When Triggered / কখন ট্রিগার হয় |
|-------|---------------------|-----------------------------------|
| `onKeyDown` | Key press down | Key is pressed down |
| `onKeyUp` | Key release | Key is released |

### Form Events / ফর্ম ইভেন্ট

| Event | Purpose / উদ্দেশ্য | When Triggered / কখন ট্রিগার হয় |
|-------|---------------------|-----------------------------------|
| `onChange` | Value change | Input value changes |
| `onSubmit` | Form submission | Form is submitted |
| `onFocus` | Focus gained | Element gains focus |
| `onBlur` | Focus lost | Element loses focus |

## Event Object Properties / ইভেন্ট অবজেক্ট প্রপার্টি

### Common Properties / সাধারণ প্রপার্টি

```jsx
function handleEvent(event) {
  console.log(event.type);        // Event type
  console.log(event.target);      // Element that triggered event
  console.log(event.currentTarget); // Element event handler is attached to
  console.log(event.preventDefault); // Prevent default behavior
  console.log(event.stopPropagation); // Stop event bubbling
}
```

### Mouse Event Properties / মাউস ইভেন্ট প্রপার্টি

```jsx
function handleMouseEvent(event) {
  console.log(event.clientX, event.clientY); // Mouse position
  console.log(event.button);                 // Which button clicked
  console.log(event.ctrlKey, event.shiftKey); // Modifier keys
}
```

### Keyboard Event Properties / কীবোর্ড ইভেন্ট প্রপার্টি

```jsx
function handleKeyboardEvent(event) {
  console.log(event.key);     // Key value
  console.log(event.code);    // Physical key code
  console.log(event.keyCode); // Numeric key code (deprecated)
  console.log(event.ctrlKey); // Ctrl key pressed
}
```

## Best Practices / সেরা অনুশীলন

### 1. Use Proper Event Types / সঠিক ইভেন্ট টাইপ ব্যবহার করুন

```jsx
// ✅ Good - Use semantic events
<button onClick={handleClick}>Click</button>
<input onChange={handleChange} />

// ❌ Avoid - Generic events
<div onClick={handleClick}>Clickable div</div>
```

### 2. Prevent Default When Needed / প্রয়োজনে ডিফল্ট আচরণ প্রতিরোধ করুন

```jsx
function handleSubmit(event) {
  event.preventDefault(); // Prevent form submission
  // Handle form data
}
```

### 3. Use Event Delegation / ইভেন্ট ডেলিগেশন ব্যবহার করুন

```jsx
// ✅ Good - Event delegation
<ul onClick={handleListClick}>
  {items.map(item => <li key={item.id} data-id={item.id}>{item.name}</li>)}
</ul>
```

## Performance Tips / পারফরমেন্স টিপস

### 1. Avoid Inline Functions / ইনলাইন ফাংশন এড়িয়ে চলুন

```jsx
// ❌ Bad - Creates new function every render
<button onClick={() => doSomething()}>Click</button>

// ✅ Good - Stable function reference
const handleClick = useCallback(() => doSomething(), []);
<button onClick={handleClick}>Click</button>
```

### 2. Debounce Expensive Operations / ব্যয়বহুল অপারেশন ডেবাউন্স করুন

```jsx
const debouncedSearch = useCallback(
  debounce((value) => performSearch(value), 300),
  []
);

<input onChange={(e) => debouncedSearch(e.target.value)} />
```

## Summary / সারসংক্ষেপ

React SyntheticEvent সিস্টেম cross-browser compatibility এবং consistent API প্রদান করে। মূল বিষয়গুলি:

1. **Event Types**: Mouse, keyboard, form, এবং touch events
2. **Event Properties**: target, type, preventDefault, stopPropagation
3. **Best Practices**: Semantic events, proper delegation, performance optimization
4. **Cross-browser**: Same API across all browsers
