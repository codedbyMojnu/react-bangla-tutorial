# DOM Elements Reference / DOM এলিমেন্ট রেফারেন্স

React-এ সব HTML DOM elements সাপোর্ট করা হয়। এই পেজে আপনি React-এ ব্যবহৃত সকল DOM elements এর comprehensive reference পাবেন।

## Quick Reference / দ্রুত রেফারেন্স

### Text Elements / টেক্সট এলিমেন্ট

| Element | Purpose / উদ্দেশ্য | Example / উদাহরণ |
|---------|---------------------|-------------------|
| `<h1>` - `<h6>` | Headings / শিরোনাম | `<h1>Main Title</h1>` |
| `<p>` | Paragraph / অনুচ্ছেদ | `<p>Text content</p>` |
| `<span>` | Inline text / ইনলাইন টেক্সট | `<span>Inline text</span>` |
| `<div>` | Block container / ব্লক কন্টেইনার | `<div>Content</div>` |
| `<strong>` | Bold text / গাঢ় টেক্সট | `<strong>Important</strong>` |
| `<em>` | Emphasized text / জোরালো টেক্সট | `<em>Emphasized</em>` |

### Form Elements / ফর্ম এলিমেন্ট

| Element | Purpose / উদ্দেশ্য | Example / উদাহরণ |
|---------|---------------------|-------------------|
| `<input>` | Input field / ইনপুট ফিল্ড | `<input type="text" />` |
| `<textarea>` | Multi-line input / মাল্টি-লাইন ইনপুট | `<textarea />` |
| `<select>` | Dropdown / ড্রপডাউন | `<select><option /></select>` |
| `<button>` | Button / বাটন | `<button>Click</button>` |
| `<form>` | Form container / ফর্ম কন্টেইনার | `<form onSubmit={handleSubmit}>` |
| `<label>` | Input label / ইনপুট লেবেল | `<label htmlFor="input">` |

## Table Elements / টেবিল এলিমেন্ট

### Table Structure / টেবিল স্ট্রাকচার

| Element | Purpose / উদ্দেশ্য | Description / বর্ণনা |
|---------|---------------------|----------------------|
| `<table>` | Table container | টেবিল কন্টেইনার |
| `<thead>` | Table header group | টেবিল হেডার গ্রুপ |
| `<tbody>` | Table body group | টেবিল বডি গ্রুপ |
| `<tfoot>` | Table footer group | টেবিল ফুটার গ্রুপ |
| `<tr>` | Table row | টেবিল রো |
| `<th>` | Header cell | হেডার সেল |
| `<td>` | Data cell | ডেটা সেল |

## React-Specific Attributes / React-নির্দিষ্ট অ্যাট্রিবিউট

### Key Differences from HTML / HTML থেকে মূল পার্থক্য

| HTML Attribute | React Attribute | Purpose / উদ্দেশ্য |
|----------------|-----------------|---------------------|
| `class` | `className` | CSS class assignment |
| `for` | `htmlFor` | Label association |
| `tabindex` | `tabIndex` | Tab order |
| `readonly` | `readOnly` | Read-only input |
| `maxlength` | `maxLength` | Maximum length |
| `onclick` | `onClick` | Click event handler |

### Event Attributes / ইভেন্ট অ্যাট্রিবিউট

| Category | Events / ইভেন্ট | Example / উদাহরণ |
|----------|------------------|-------------------|
| Mouse | `onClick`, `onMouseOver`, `onMouseOut` | `onClick={handleClick}` |
| Keyboard | `onKeyDown`, `onKeyUp`, `onKeyPress` | `onKeyDown={handleKeyDown}` |
| Form | `onSubmit`, `onChange`, `onFocus`, `onBlur` | `onChange={handleChange}` |
| Touch | `onTouchStart`, `onTouchEnd`, `onTouchMove` | `onTouchStart={handleTouch}` |

## Best Practices / সেরা অনুশীলন

### 1. Semantic HTML Usage / সিম্যান্টিক HTML ব্যবহার

```jsx
// ✅ Good - Semantic structure
<article>
  <header>
    <h1>Article Title</h1>
    <time>Published on March 1, 2024</time>
  </header>
  <main>
    <p>Article content...</p>
  </main>
  <footer>
    <p>Author: John Doe</p>
  </footer>
</article>

// ❌ Bad - Non-semantic divs
<div>
  <div>
    <div>Article Title</div>
    <div>Published on March 1, 2024</div>
  </div>
  <div>
    <div>Article content...</div>
  </div>
</div>
```

### 2. Accessibility Attributes / অ্যাক্সেসিবিলিটি অ্যাট্রিবিউট

```jsx
// ✅ Good - Accessible form
<form>
  <label htmlFor="email">Email Address:</label>
  <input
    type="email"
    id="email"
    aria-describedby="email-help"
    required
  />
  <small id="email-help">We'll never share your email</small>
  
  <button type="submit" aria-label="Submit contact form">
    Submit
  </button>
</form>

// ✅ Good - Accessible image
<img
  src="chart.png"
  alt="Sales increased by 25% in Q1 2024"
  role="img"
/>
```

### 3. Form Validation / ফর্ম ভ্যালিডেশন

## Common Pitfalls / সাধারণ ভুল

### 1. Incorrect Attribute Names / ভুল অ্যাট্রিবিউট নাম

```jsx
// ❌ Wrong - HTML attributes in JSX
<div class="container" for="input">
  <input tabindex="1" readonly />
</div>

// ✅ Correct - JSX attributes
<div className="container" htmlFor="input">
  <input tabIndex="1" readOnly />
</div>
```

### 2. Missing Key Props / অনুপস্থিত Key Props

```jsx
// ❌ Wrong - Missing keys in lists
{items.map(item => <li>{item.name}</li>)}

// ✅ Correct - Proper keys
{items.map(item => <li key={item.id}>{item.name}</li>)}
```

### 3. Inline Event Handlers / ইনলাইন ইভেন্ট হ্যান্ডলার

```jsx
// ❌ Wrong - New function on every render
<button onClick={() => console.log('clicked')}>
  Click me
</button>

// ✅ Better - Stable reference
const handleClick = useCallback(() => {
  console.log('clicked');
}, []);

<button onClick={handleClick}>Click me</button>
```

## Performance Considerations / পারফরমেন্স বিবেচনা

### 1. Avoid Unnecessary Re-renders / অপ্রয়োজনীয় রি-রেন্ডার এড়িয়ে চলুন

```jsx
// ✅ Good - Memoized component
const ExpensiveComponent = React.memo(({ data }) => {
  return (
    <div>
      {data.map(item => (
        <div key={item.id}>{item.name}</div>
      ))}
    </div>
  );
});
```

### 2. Optimize Event Handlers / ইভেন্ট হ্যান্ডলার অপ্টিমাইজ করুন

```jsx
// ✅ Good - Event delegation
<div onClick={handleContainerClick}>
  {items.map(item => (
    <button key={item.id} data-id={item.id}>
      {item.name}
    </button>
  ))}
</div>
```
