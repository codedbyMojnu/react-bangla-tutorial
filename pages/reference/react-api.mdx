### useEffect

**Signature:** `useEffect(effect, dependencies?)`

**Purpose:** Side effects handle ‡¶ï‡¶∞‡¶æ (API calls, subscriptions, timers)  
**‡¶â‡¶¶‡ßç‡¶¶‡ßá‡¶∂‡ßç‡¶Ø:** Side effects ‡¶™‡¶∞‡¶ø‡¶ö‡¶æ‡¶≤‡¶®‡¶æ ‡¶ï‡¶∞‡¶æ

```javascript

  const [user, setUser] = useState(null);

  // Effect without dependencies (runs after every render)
  useEffect(() => {
    document.title = `User: ${user?.name || 'Loading...'}`;
  });

  // Effect with dependencies (runs when userId changes)
  useEffect(() => {
    fetchUser(userId).then(setUser);
  }, [userId]);

  // Effect with cleanup (unsubscribe on unmount)
  useEffect(() => {
    const subscription = subscribe(userId);
    
    return () => {
      subscription.unsubscribe();
    };
  }, [userId]);

  // Effect with empty dependencies (runs only once)
  useEffect(() => {
    const timer = setInterval(() => {
      console.log('Timer tick');
    }, 1000);

    return () => clearInterval(timer);
  }, []);

  return <div>{user ? user.name : 'Loading...'}</div>;
}
```

### useReducer

**Purpose:** Complex state management alternative to useState  
**‡¶â‡¶¶‡ßç‡¶¶‡ßá‡¶∂‡ßç‡¶Ø:** ‡¶ú‡¶ü‡¶ø‡¶≤ state management ‡¶è‡¶∞ ‡¶ú‡¶®‡ßç‡¶Ø useState ‡¶è‡¶∞ ‡¶¨‡¶ø‡¶ï‡¶≤‡ßç‡¶™

```javascript
  switch (action.type) {

    case 'decrement':
      return { count: state.count - 1 };
    case 'reset':
      return initialState;
    default:
      throw new Error();
  }
}

  const [state, dispatch] = useReducer(reducer, initialState);

  return (
    <div>
      Count: {state.count}
      <button onClick={() => dispatch({ type: 'increment' })}>+</button>
      <button onClick={() => dispatch({ type: 'decrement' })}>-</button>
      <button onClick={() => dispatch({ type: 'reset' })}>Reset</button>
    </div>
  );
}
```

**Purpose:** DOM elements access ‡¶ï‡¶∞‡¶æ ‡¶è‡¶¨‡¶Ç mutable values store ‡¶ï‡¶∞‡¶æ

```javascript
    };
  }, []);

      <input ref={inputEl} type="text" />
      <button onClick={onButtonClick}>Focus the input</button>
    </div>
  );
}
```
