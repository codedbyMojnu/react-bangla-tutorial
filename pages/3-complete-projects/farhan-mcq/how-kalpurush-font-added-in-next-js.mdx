### **Step 3: Use `next/font/local` (Next.js 13+)**

This is the **official & optimized** way.

üìÅ `app/layout.js` (or `layout.tsx`)

```js

  src: "../public/fonts/Kalpurush.ttf",
  variable: "--font-kalpurush",
  weight: "400",
});

  return (
    <html lang="bn" className={kalpurush.variable}>
      <body>{children}</body>
    </html>
  );
}
```
