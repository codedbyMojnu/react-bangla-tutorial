# Components | কম্পোনেন্ট

React components হচ্ছে UI এর building blocks। এরা independent, reusable pieces of code যা complex UI তৈরি করতে ব্যবহৃত হয়।

## What are Components? / কম্পোনেন্ট কি?

Component হল একটি JavaScript function বা class যা:
- **Props** accept করে (input হিসেবে)
- **React elements** return করে (যা screen এ render হয়)
- **Reusable** এবং **composable**

```jsx
// Simple Component
function Welcome() {
  return <h1>Hello, World!</h1>;
}
```

## Types of Components / কম্পোনেন্টের ধরন

### 1. Function Components / ফাংশন কম্পোনেন্ট

Modern React এ preferred approach। Simple এবং hooks ব্যবহার করতে পারে।

### 2. Class Components / ক্লাস কম্পোনেন্ট

Legacy approach কিন্তু এখনও ব্যবহৃত হয়। Lifecycle methods এবং state management এর জন্য।

## Component Best Practices / কম্পোনেন্ট সেরা অনুশীলন

### 1. Single Responsibility / একক দায়িত্ব

```jsx
// ❌ Bad - Multiple responsibilities
function UserDashboard({ user }) {
  return (
    <div>
      <img src={user.avatar} alt={user.name} />
      <h1>{user.name}</h1>
      <p>{user.email}</p>
      <div>
        {user.posts.map(post => (
          <div key={post.id}>
            <h3>{post.title}</h3>
            <p>{post.content}</p>
            <span>{post.date}</span>
          </div>
        ))}
      </div>
    </div>
  );
}

// ✅ Good - Separated responsibilities
function UserProfile({ user }) {
  return (
    <div>
      <img src={user.avatar} alt={user.name} />
      <h1>{user.name}</h1>
      <p>{user.email}</p>
    </div>
  );
}

function PostList({ posts }) {
  return (
    <div>
      {posts.map(post => (
        <Post key={post.id} post={post} />
      ))}
    </div>
  );
}

function Post({ post }) {
  return (
    <div>
      <h3>{post.title}</h3>
      <p>{post.content}</p>
      <span>{post.date}</span>
    </div>
  );
}
```

### 2. Proper Naming / সঠিক নামকরণ

```jsx
// ❌ Bad names
function Comp1({ data }) { /* ... */ }
function thing({ user }) { /* ... */ }
function handleStuff() { /* ... */ }

// ✅ Good names
function UserProfileCard({ user }) { /* ... */ }
function ProductList({ products }) { /* ... */ }
function handleUserLogin() { /* ... */ }
```

### 3. Default Props / ডিফল্ট Props

```jsx
function Button({ 
  children, 
  variant = 'primary', 
  size = 'medium', 
  disabled = false,
  onClick = () => {}
}) {
  return (
    <button 
      className={\`btn btn-\${variant} btn-\${size}\`}
      disabled={disabled}
      onClick={onClick}
    >
      {children}
    </button>
  );
}
```

## Summary / সারসংক্ষেপ

Components হল React এর core concept। এই অধ্যায়ে আমরা শিখেছি:

1. **Function vs Class Components**: Modern React এ function components preferred
2. **Component Composition**: ছোট components দিয়ে complex UI তৈরি
3. **Props vs Children**: কখন কোনটি ব্যবহার করবেন
4. **Best Practices**: Clean, maintainable components লেখার নিয়ম
