## React.js-এর ইতিহাস

২০১১ সালে ফেসবুকের **Jordan Walke** React তৈরি করেন। তিনি PHP-এর Component-Based Framework "XHP" থেকে অনুপ্রাণিত হয়ে এটি ডেভেলপ করেন। React প্রথম ফেসবুকের নিউজফিডে এবং পরে ২০১২ সালে ইনস্টাগ্রামে ব্যবহার করা হয়।

২০১৩ সালে ফেসবুক React.js-কে ওপেন সোর্স ঘোষণা করে। এরপর থেকে React দ্রুত জনপ্রিয়তা পায় এবং বিভিন্ন বড় কোম্পানি ও ডেভেলপারদের মধ্যে ছড়িয়ে পড়ে।

## রিয়্যাক্ট কী?

React হলো JavaScript-এর একটা লাইব্রেরি যেটা দিয়ে খুব সহজে UI (User Interface) বানানো যায়। আর UI মানে যা ব্যবহারকারী ব্রাউজারে দেখে এবং ব্যবহার করে।

> যেমন _Facebook NewsFeed_, _ইউটিউবের ভিডিও লিস্ট_ বা _ইনপুট ফর্ম_ — এই সব কিছুই UI।

**React একসাথে কাজ করে ওয়েব, মোবাইল—সব প্ল্যাটফর্মেই।**

### ১। ছোট ছোট component জুড়ে দিয়ে ইউজার ইন্টারফেস বানানো

React-এ আমরা কিন্তু সম্পূর্ণ ওয়েবসাইটকে একবারে বানাই না। React বলে, **"পুরো জিনিস একসাথে বানাতে যাও না ভাই, টুকরা টুকরা করে বানাও!"** প্রতিটা ছোট টুকরাকে বলে বলি **Component**।

> এইভাবে কম্পোনেন্ট-ভিত্তিক কাজ করার সবচেয়ে বড় সুবিধা হলো, <span style={{borderBottom: "1px dashed black", padding: "1px"}}>কোড থাকে খুব গোছানো এবং টিমের সাথে কাজ করা সহজ হয়। প্রত্যেকে টিম মেম্বার তার নিজের কম্পোনেন্ট নিয়ে কাজ করতে পারে।</span>

ধরেন, আপনি YouTube-এর মতো একটি পেজ বানাচ্ছেন। সেখানে থাকতে পারে:

- `<Thumbnail />` — ভিডিও থাম্বনেল
- `<LikeButton />` — লাইক বাটন
- `<Video />` — পুরো ভিডিও ব্লক
- `<Comment />` - কমেন্ট

এই Component গুলোকে একসাথে জোড়া দিয়ে বানিয়ে ফেলা যায় একটা আস্ত বড় অ্যাপ, মানে যেটাকে "PRODUCT" বলছি।

```jsx
// Video.js
function Video({ video }) {
  return (
    <div>
      <VideoThumbnail image={video.thumbnail} />
      <a href={video.url}>
        <VideoTitle title={video.title} />
        <p>{video.description}</p>
      </a>
      <LikeButton />
    </div>
  );
}
```

### ২। JSX — যেখানে কোড আর ডিজাইন একসাথে লিখি

React কম্পোনেন্ট আদতে একটি জাভাস্ক্রিপ্ট ফাংশন। React-এ আমরা যে কোড লিখি সেটা JSX। দেখতে HTML এর মতো মনে হলেও, কিন্তু এর ভিতর JS-এর সমস্ত পাওয়ার আছে।

```jsx
return <h1>{heading}</h1>;
```

এখানে `{heading}` হচ্ছে JavaScript expression, আর `<h1>` হচ্ছে JSX tag।

<span style={{ borderBottom: "1px dashed black", padding: "1px" }}>
  React এটাকে জাভাস্ক্রিপ্টে রূপান্তর করে, যা ব্রাউজার বুঝতে পারে।
</span>
পর্দার আড়ালে এটা JSX, মূলত ব্রাউজারে যাওয়ার আগে নিচের কোডে রূপ নেয়:

```js
React.createElement("h1", null, "Welcome!");
```

JSX ব্যবহার কি, যাতে কোড ক্লিন থাকে, সহজে বোঝা যায় এবং মেইনটেইন করা যায়।

<hr />
ব্যাপারটা হলো, JSX আপনাকে HTML-এর ভেতরেই জাভাস্ক্রিপ্টের লেখার সুযোগ
দেয়। এর ফলে ডিজাইন (মার্কআপ) আর লজিক এক জায়গায় থাকতেছে, যা ম্যানেজ করা
অনেক সহজ করে দেয়।

**উদাহরণ:**

```jsx
// VideoList.js
function VideoList({ videos, emptyHeading }) {
  const count = videos.length;
  let heading = emptyHeading;

  if (count > 0) {
    const noun = count > 1 ? "videos" : "video";
    heading = count + " " + noun;
  }

  return (
    <section>
      <h1>{heading}</h1>
      {/* এখানে .map() ব্যবহার করে প্রতিটা ভিডিওর জন্য একটি কম্পোনেন্ট দেখানো হচ্ছে */}
      {videos.map((video) => (
        <Video key={video.id} video={video} />
      ))}
    </section>
  );
}
```

> **প্রশ্ন:** `<Welcome/>` কম্পোনেন্টের `<h1>` ট্যাগটা কি আসল HTML?

### ৩। যেখানেই দরকার, সেখানেই ইন্টারঅ্যাকশন যোগ

নিচের উদাহরণে, আপনি যখন সার্চ বক্সে কিছু টাইপ করবেন, React সাথে সাথে ভিডিও লিস্ট থেকে ফিল্টার করে দেখাবে।

> **নোট:** নিচের কোডটি এখন পুরোপুরি বোঝার দরকার নেই। শুধু একবার চোখ বুলিয়ে রাখেন, যে কীভাবে একটি ইন্টারঅ্যাকটিভ লিস্ট বানানো যায়।

```jsx
// SearchableVideoList.js

// ... (এখানে ভিডিওর ডেটা এবং অন্যান্য কম্পোনেন্ট থাকবে) ...

function SearchableVideoList({ videos }) {
  const [searchText, setSearchText] = useState("");
  const foundVideos = filterVideos(videos, searchText); // ফিল্টার করা ভিডিও

  return (
    <>
      <SearchInput value={searchText} onChange={setSearchText} />
      <VideoList
        videos={foundVideos}
        emptyHeading={`"${searchText}" দিয়ে কিছু পাওয়া যায়নি`}
      />
    </>
  );
}
```

### ৬। আপনার পুরোনো প্রজেক্টেও React যুক্ত করা সম্ভব

React শেখার জন্য আপনাকে নতুন অ্যাপ বানাতেই হবে এমন না।
চাইলেই আপনার পুরোনো ওয়েবসাইটে React কম্পোনেন্ট যুক্ত করতে পারেন:

```html
<nav id="navigation"></nav>
```

```js
import { createRoot } from "react-dom/client";

function NavigationBar() {
  return <h1>Hello from React!</h1>;
}

const root = createRoot(document.getElementById("navigation"));
root.render(<NavigationBar />);
```

## 🔹 React-এর মূল কিছু ফিচার এক নজরে:

- ✅ **Component-based Structure** — ছোট ছোট ইউনিটে কোড ভাগ
- ⚡ **Virtual DOM** — DOM ম্যানিপুলেশন দ্রুত হয়
- 🎯 **Declarative UI** — কী চাই সেটাই বলে দেন, React ব্যবস্থা নেবে
- 🔄 **State & Props** — কম্পোনেন্টে ডেটা ব্যবস্থাপনা
- 🔌 **Hooks** — যেমন `useState`, `useEffect` দিয়ে ফাংশনাল কম্পোনেন্টে ম্যাজিক
